# Portfolio Next.js

## Description
Ce projet est un portfolio web développé avec Next.js.  
Il permet de gérer des projets et des témoignages avec authentification.

---

## Fonctionnalités
- Login / Inscription
- Gestion des projets (liste et détail)
- Gestion des témoignages (ajout et modification)
- Protection des routes avec middleware
- Utilisation de Redux Toolkit

---

## Technologies
- Next.js
- React
- Redux Toolkit
- Sequelize
- SQLite
- JWT
